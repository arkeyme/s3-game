SELECT the right way to solve it :)

S3 Select is designed to pull out only the data you need from an object, which can improve the performance and reduce the cost.

S3 Select filters the content of an Amazon S3 object based on SQL statement. In the request, you must specify a data format (JSON, CSV, or Apache Parquet). Amazon S3 uses this format to parse object data into records and returns only records that match the specified SQL expression. You must also specify the data serialization format for the response.
