Keeping buckets open is not the best practice. Access to AWS resources can be controlled with access keys. 

Hint 1
Access key consist of two parts: an access key ID and a secret access key. 
Like a user name and password, you must use both the access key ID and secret access key together to authenticate your requests.
Important: Manage your access keys as SECURELY as you do your user name and password!!!
